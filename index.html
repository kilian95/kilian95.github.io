Car Share

This project was created during the final year of University. The aim was to create a 2 sided platform where users could share lifts. Created using Ruby on Rails, Javascript, Bootstrap as the front end framework and Postgres as the chosen database. The web app had a range of features including user authentication, a real time messaging system implemented using websockets, a review and notification system and admin capability. The app was deployed using the cloud platform Heroku, with user uploaded images being stored using AWS's EC2 service.

Graffiti Near Me

This project, which I worked on in my spare time had the aim of documenting the location of street art in Belfast. Using Ruby on Rails to build the app, users could upload images of street art and through the use of the Google Maps API, they could then mark the location so other users could see the art work in person. For this project, my main objectives were to create a single page aplication which was intuitive and simple to use.
